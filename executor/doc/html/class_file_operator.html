string FileOperator::removeParentheses ( std::string  s)

Remove parentheses in a string.

Parameters:
sString that contains parentheses
Returns:
The string s without parentheses

The different steps are:

  • Find the position pos2 of the closing parenthesis ")"
    • Use pos2 and remove the closing parenthesis from s: use the string::erase function
  • Find the position pos2 of the opening parenthesis "("
    • Keep the string that starts after pos2: use the string::substr function
