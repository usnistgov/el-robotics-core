void KittingPlan::parseLinePlanInstance ( string  action)

Parse an action and separate the action name to its parameters The action name and parameters are put in a map <name,vector<string>>

Parameters:
actionAction to be parsed

The different steps are:

  • Find the position of the closing parenthesis ")"
  • Remove it from s (erase)
  • Find the position of the opening parenthesis "("
  • Use its position to keep the rest of the string s (substr)

Example:

  • Action to execute: (attach-eff robot_1 tray_gripper tray_gripper_holder)
  • Result: <attach-eff, <robot_1,tray_gripper,tray_gripper_holder>>
