To add modified files to be staged for committing:

   git add <file>

To add new files to be tracked:

   git add <files>

To fully commit and check back changes:

   git add <files>
   git commit
   git push

To do the 'add' and 'commit' all at once:

   git commit -a
   git push

To see the files that are modified, and untracked:

   git status

To list all files currently tracked by Git: 

   git ls-files
