Detailed Description

The CLinkReader is a class that use the tf package from ROS to return latest pose of a link. Uses tf to find the latest time and base_link and the desired link have changed. With the latest time it returns translation and orientation (quat) of the link. tf keeps track of a tree of coordinate frames. This tree changes over time, and tf stores a time snapshot for every transform (for up to 10 seconds by default).
