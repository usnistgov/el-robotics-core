class  RobotStatus
 The RobotStatus is a thread that reads the status of the robot and updates the world model. The RobotStatus is a separate thread that reads the robot status using ROS communication mechanisms and updates the controller world model based on these values. Currently, it uses an instance of the class JointReader to read joint values from the controller. It uses a Kinematics pointer reference to compute the current robot pose using the forward kinematics (FK) routine. It also uses a CrclDelegate pointer reference to update the status reported by CRCL. More...
